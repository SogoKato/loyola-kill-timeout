# LOYOLAタイムアウト回避
LOYOLAの画面上部にある時計ボタンを1分おきに自動でクリックするスクリプトです。
サーバーの負荷を考慮し、良識的な範囲で使い、ログアウトすることを心がけてください。

# 対応ブラウザ
Chrome, Chromium版Microsoft Edge, Firefox

# インストール
Chrome Web Storeの審査が通らなかったので（審査担当者がLOYOLAにログインできないためだと思われる）、ご自身でダウンロードしてインストールしてください。

[ダウンロード](https://github.com/norocchi/loyola-kill-timeout/blob/master/dist/loyola-kill-timeout.zip?raw=true)
ZIPファイルを解凍してください。

Chrome / Chromium版Microsoft Edge:
1. [chrome://extensions/](chrome://extensions/)を開きます。
2. 「デベロッパー モード」または「開発者モード」をオンにする。
3. 「パッケージ化されていない拡張機能を読み込む」または「展開して読み込み」を押し、該当のフォルダを選択します。

Firefox:
1. [about:debugging](about:debugging)を開きます。
2. 「このFirefox」タブを選択します。
3. 「一時的なアドオンを読み込む…」を押し、該当のフォルダを選択します。これはブラウザを閉じるまで有効です。
